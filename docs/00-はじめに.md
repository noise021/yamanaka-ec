# 00. はじめに

## このマニュアルについて

このマニュアルは、Next.jsを使用したECサイトを一から開発し、テンプレート化するまでの手順をまとめたものです。

実際にコードを書きながら進めるため、開発記録としても利用できます。

---

## 開発環境

- Next.js(App Router)
- TypeScript
- Tailwind CSS
- Supabase
- Stripe
- GitHub
- Vercel

---

## このマニュアルの進め方

各章ごとに次の順番で進めます。

1. 作る機能を確認
2. コードを書く
3. 動作確認
4. GitHubへ反映
5. 本書(docs)を更新

---

## 完成目標

完成すると以下の機能を持つECサイトになります。

- 商品一覧
- 商品詳細
- カート
- 会員登録
- ログイン
- Stripe決済
- 管理画面
- SEO対策
- AI機能
